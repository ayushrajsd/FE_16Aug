CSS Specificity

CSS specificity is a set of rules that determine which CSS styles should be applied to an HTML element when there are multiple conflicting styles targeting the same element. - the more specific style gets applied

Rule A: Applies a red text color to all paragraphs. Rule B: Applies a blue text color to paragraphs with a class of "highlight." Rule C: Applies an italic font to paragraphs with an ID of "special-paragraph."

calculating specificity

specificity is set of triplets ( a, b, c )

a- IDs

b - classes

c - elements

#apple>p => 101

.fruits.banana>span=>021
