Web Vitals

  • TTFB - time to first byte: the time taken for the first byte to reach client from server
  • Largest contentful paint(core web vital) - time taken to paint the largest image/ section on UI
  • CLS( core web vital) - cumulative layout shift: when content shifts after page loads
  • FID (core web vital) - first input delay - the delay after which your web page becomes interactive for user
